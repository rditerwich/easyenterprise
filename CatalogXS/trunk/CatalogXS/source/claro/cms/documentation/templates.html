Templates

A site consists of top-level templates and templates that are included from other templates. There are three commands that deal with template inclusion:
elementdescription
<template:include template="name">Includes another template
<template:define template="name">Defines a template that can be included by nested templates
<template:content">Renders the default content of the enclosing template:include

Usage

Use template:include to include another template. 
<div>
  <template:include template="other-template">
    content
  </template:include>
</div>
The content body is optional. It is rendered when the included template was not found, or when template:content is used inside the included template.

Default template content

An template:include element can have content. This content is rendered when the included template was not found. This way, one can use template:include as an extension mechanism: 
<head>
  <template:include template="custom-scripts">
    <script src="/json.js" type="text/javascript" />
  </template:include>
</head>


Output:

<head>
  <script src="/json.js" type="text/javascript" />
</head>



Include default content explicitly


When writing the template custom-scripts.html, one can explicitly include the default content using template:content:


<script src="/myscript.js" type="text/javascript" />
<template:content />


Output:

<head>
  <script src="/myscript.js" type="text/javascript" />
  <script src="/json.js" type="text/javascript" />
</head>
